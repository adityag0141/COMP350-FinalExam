Application Design Issues

Issues in the design and suggestions

  • 23rdApril, 2022

Navbar

  • The navbar is designed in a nonstandard style. There should be buttons instead of links as buttons give a more appealing look.
  • A Multiplayer link is highlighted which is giving vague effect on the whole webpage
  • About section redirects to the new pop-up. Whereas on the footer there is another About section that has links to a whole new different website that has given the information about the company associated with the development of this game.

Main Body

  • They have mentioned the select the difficulty, but I was not able to find any option to select the difficulty level of the game.
  • The maps are organized in the cards, but all the countries are displayed in the main body which are filling up the whole content of the main body.
  • The spaces between each row of cards have been used by Google AdSense to show up the advertisements which do not make any sense.

Footer

  • The heading Inspired By, New Releases, share us, and about should be the buttons of the navigation bar. New Releases button again contains Streaks and Multiplayer option which is making it very confusing.

Gameplay

  • From my point of view, users should be given the option of choosing the city when answering. As the name City Guesser, it means we must guess the city. But here we have to mark the locations on the map.
  • While the video is playing, there should be a media seek so that the user can play the video at timestamps. As this game is all about guessing the cities through vehicle number plates, monuments, etc. seek is necessary so that the user can have detailed attention.
