Interface

Description of menus, UI, etc.

  • 21stApril, 2022

Header

  • The navigation bar contains three buttons About, Streaks, and Multiplayer.
  • About button redirects to the page which answers a few frequently asked questions.
  • Streaks contain two modes, Country Streak and Radius Streak.
  • The Multiplayer button allows the user to connect to a private room for playing with other members of the user can create his private room.

Main Body

  • Different cards of places where users can select any of them and can guess the location on the map..
  • In the middle of this section, there are two categories of the game. One is Streaks, which redirects to the same page as it was doing in the Streaks option in the navigation bar.
  • Streaks contain two modes, Country Streak and Radius Streak.
  • The Challenge category has two challenges. The first one is the No moving challenge, and the other is Time Limit Challenge.

Footer

  • There are 4 menus in the footer section i.e., Inspired By, New Releases, Share Us, and About.
  • Inspired By menu contains links to a different website from which the City Guesser has taken inspiration. GeoGuessr is one the same game as City Guesser. It has used cards from W3 schools and the sample code with CSS is available in the Modern Travel Cards link.
  • In the New Releases menu, there are pretty much the same options that were available in the navigation bar. The website virtualvacations.us has created virtual Walking and Driving Tour feature.
  • Share Us menu contains all the links to social media handles of the website like Facebook, Twitter, Pinterest, and LinkedIn.
