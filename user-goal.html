User-Goal

Sequence of Steps to achieve user-goal

  • 22ndApril, 2022

Steps

  1. As soon as the user visits the webpage, the user sees different countries, cities, and monuments. Users can start playing the game by clicking on the "Guess" button

  2. The user can click on the multiplayer button to play with other people online. It also has the option to create your private room, or the user can join in the previously created random room. If the user is joining the pre-existing room, then the user should have the unique for entering the room.

  3. Another option is of Streaks Challenge. There are two modes in this challenge. First, the user can select the time limit. After that timer will start and the user must guess the countries in the given time. Another mode is radius mode. In this user can select the radius range from his current location and the location will be displayed within that selected range and the user must guess that location.

  4. On the middle of the home page, the UI also gives us the option of Challenge Mode. There are two modes in that. First is the no-moving mode. In that, a still image will be displayed, and the user must guess the location. The second mode is the Time Limit Challenge mode. In that user can set the time limit in seconds and can select the country or city according to his/her choice. A timer will begin above the video and the video will play according to the time limit.

  5. There are some video controls available. The clock icon allows adjusting the speed of the video. There is also an audio control bar. Users can restart the video by clicking on the restart icon. Reverse icon reverses the video by 10 sec.

  6. When the user starts guessing the location, there are map zoom-in zoom-out controls. The user has the choice of selecting the map language in English or the native language of the respective countries. The user pins the location on the map, and in response gets how close he marked that location in miles.
